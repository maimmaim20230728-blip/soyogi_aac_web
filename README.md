# そよぎ式AACアプリ

はなすことが むずかしい ひとのための、**無料**のコミュニケーションツール（AAC = 補助代替コミュニケーション）です。

- 📱 インストール不要（ブラウザで動く・PWAとしてホーム画面にも置ける）
- 💰 完全無料・広告なし・登録不要
- 🔌 オフラインで動作
- 🔒 書いた内容は端末の中にだけ保存。**どこにも送信しません**
- 🗣 タップした言葉を読み上げ（Web Speech API）
- 📄 声が出ない端末でも「みせる」画面（でか文字表示）で伝えられる

## 画面

| 画面 | 役割 |
|---|---|
| はなす | カードをタップして文を作り、読み上げる |
| きんきゅう | 「たすけてください」等を1タップで大きく読み上げ |
| はい・いいえ | 巨大な2択＋わからない |
| みせる | でか文字を相手に見せる（筆談ボード） |
| せってい | 言語・文字/カードの大きさ・声・じぶんのカード |

## 思想

高額なAAC機器・アプリを買えない人が世界中にいます。このアプリは公共財です。
自由にフォークして、あなたの言語・あなたの現場に合わせて作り替えてください。

制作: 介護と支援の相談どころ「そよぎ」
